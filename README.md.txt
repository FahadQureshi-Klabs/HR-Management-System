# 🏢 HR Management System

A complete Human Resource Management System with modern UI, dark mode, and full employee management features.

## ✨ Features


- 🔐 **Authentication System** - HR Admin and Employee login
- 👥 **Employee Management** - Add, edit, delete employees
- 📅 **Attendance Tracking** - Check-in/Check-out with late marking
- 📝 **Leave Management** - Apply for leaves (Sick/Casual/Annual)
- 🏠 **WFH Requests** - Work from home request system
- 📢 **Announcements** - Company-wide announcements
- 🌓 **Dark Mode** - Toggle between light/dark themes
- 💾 **Local Storage** - All data persists in browser
- 📊 **Dashboard Stats** - Real-time analytics
- 🎨 **Modern UI** - Gradient designs, smooth animations
